# chd_location_draw
A cpp program for drawing chd game portal and object.
![demo](images/demo.png)
## Build
```
mkdir build
cmake ..
make
```
will generate .dll file and .exe injection file.
## Run
Run the game and click 'inject.exe'.
Just enjoy it.
